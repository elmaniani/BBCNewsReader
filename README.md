# BBC News Reader

CST2335 Final Project Android application.

Repository: https://github.com/elmaniani/BBCNewsReader

## Overview

BBC News Reader loads headlines from the BBC RSS feed and displays them in a mobile interface. Users can search headlines, open article details, save favourite articles, delete saved items, and view application information.

## Main features

- BBC RSS feed download using AsyncTask
- Article list using ListView
- Article details activity with a fragment
- Favourite articles saved with SQLite
- Delete option for saved favourites
- Toolbar and navigation drawer
- Toast, Snackbar, ProgressBar, Button, EditText, SharedPreferences, and AlertDialog
- English and Canadian French resources
